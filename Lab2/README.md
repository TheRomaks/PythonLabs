# Приложение для расчета статистики игроков, команд и матчей

Данное приложение позволяет рассчитывать статистику игроков, команд и матчей. В нем абстрактный класс 'Statistics', который содержит общий метод для подсчёта статитики. Данный класс наследуют следующие классы: 'Player', 'Team' и 'Match'

## Инструкция по работе с кодом
Для запуска основного меню выполните следующие шаги:
1. Установите необходимые библиотеки (`pip install -r requirements.txt`).
2. Скопируйте файл с кодом в ваш проект.
3. Запустите приложение с помощью команды `python main.py`.

## Справка по классам в коде
### Абстрактный класс Statistics
Данный класс определяет общие методы для всех остальных классов (в моем случае один метод для 3х классов):
- `calculate_statistics`: Этот метод объявлен с помощью декоратора @abstractmethod, что означает, что он должен быть реализован в любом классе, наследующем от Statistics. Сам метод в абстрактном классе не имеет реализации.

### Класс Player
Класс Player представляет собой объект, описывающий игрока в спортивной статистике. Он наследуется от абстрактного класса Statistics и включает в себя следующие атрибуты и методы:
- 'name': Имя игрока. Это свойство обеспечивает проверку типа данных, garantируя, что имя является строкой.
- 'goals': Количество голов, забитых игроком. Это свойство проверяет, что значение является неотрицательным целым числом.
- 'assists': Количество ассистов, совершенных игроком. Это свойство также проверяет, что значение является неотрицательным целым числом.
- __init__(name: str, goals: int = 0, assists: int = 0): Конструктор класса, инициализирующий атрибуты name, goals, и assists.
calculate_statistics(): Метод, возвращающий словарь с статистикой игрока, включающий имя, количество голов и ассистов.
- __eq__(other): Метод для сравнения двух объектов класса Player по их атрибутам.
- __str__(): Метод, возвращающий строковое представление объекта, содержащее имя игрока, количество голов и ассистов.

```python
    player = Player("Иван Иванов", 10, 5)
```

### Класс Team
Класс Team представляет собой объект, описывающий спортивную команду. Он наследуется от абстрактного класса Statistics и включает в себя следующие атрибуты и методы:
- 'name': Название команды. Это свойство обеспечивает проверку типа данных, гарантируя, что название является строкой
- 'players': Список игроков в команде. Это свойство проверяет, что все элементы списка являются экземплярами класса Player.
- __init__(name: str, players: List[Player] = None): Конструктор класса, инициализирующий атрибуты name и players.
- add_player(player: Player): Метод для добавления игрока в команду.
- remove_player(player: Player): Метод для удаления игрока из команды.
- calculate_statistics(): Метод, возвращающий словарь с статистикой команды, включающий название команды и общее количество голов, забитых всеми игроками.
- __len__(): Метод, возвращающий количество игроков в команде.
- __str__(): Метод, возвращающий строковое представление объекта, содержащее название команды и список ее игроков.

```python
    team = Team("Моя Команда")
```
### Класс Match
Класс Match представляет собой объект, описывающий матч или игру между двумя спортивными командами. Он наследуется от абстрактного класса Statistics и включает в себя следующие атрибуты и методы:
- team1: Объект класса Team, представляющий первую команду. Это свойство обеспечивает проверку типа данных, garantируя, что значение является экземпляром класса Team.
- team2: Объект класса Team, представляющий вторую команду. Это свойство также проверяет, что значение является экземпляром класса Team.
- score1: Счет первой команды. Это свойство проверяет, что значение является неотрицательным целым числом.
- score2: Счет второй команды. Это свойство также проверяет, что значение является неотрицательным целым числом.
- __init__(team1: Team, team2: Team, score1: int = 0, score2: int = 0): Конструктор класса, инициализирующий атрибуты team1, team2, score1, и score2.
- calculate_statistics(): Метод, возвращающий словарь с статистикой матча, включающий статистику обеих команд и окончательный счет.
- __str__(): Метод, возвращающий строковое представление объекта, содержащее названия команд и их счет.

```python
    team1 = Team("Команда А")
    team2 = Team("Команда Б")

    match = Match(team1, team2, 3, 2)
```
